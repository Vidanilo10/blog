### Navigation Bar
- **Hover Effect**: Text color changes to white

- **Cursor**: Pointer

3. **Whitespace**: Don't overcrowd
4. **Contrast**: Black text on beige reads well
5. **Hierarchy**: Clear visual importance
6. **Responsiveness**: Looks good on all sizes

### Photography

- Use high-quality images for movies
- Consistent image aspect ratios
- Compress images for web (TinyPNG)
- Use CDN for better loading

### Content

- Keep descriptions concise
- Use consistent tone
- Check spelling and grammar
- Update regularly
